What is TypePaste?

TypePaste is a keyboard extension you can use to set four Snippets of text, code, or whatever you choose!

You set a Snippet, save it, then press the button on the TypePaste Device to paste it to your connected computer!

To set a Snippet, simply click the "Edit" button on one of the four Snippets, enter your text (changing the name if you please), then hit save.
